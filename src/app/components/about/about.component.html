Frequently asked questions

How does searching work?

Images in Mitra are indexed using episode subtitles (closed caption text), which means that you can only search using words that were actually spoken during the episode and not things like the name of a character or name of the episode etc.

What seasons are available?

All 10 seasons are available.

Technical details?

Mitra's front-end is built in Angular and Bootstrap and back-end is built in Julia. The image data is stored in a SQLite database.
