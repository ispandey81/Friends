Frequently asked questions

How does searching work?

Images in Friends are indexed using episode subtitles (closed caption text), which means that you can only search using words that were actually spoken during the episode and not things like the name of a character or name of the episode etc.

What seasons are available?

All 10 seasons.

Technical details?

Friends's front-end is built in Angular and Bootstrap and back-end is built in Julia. The back-end services are deployed on render. The image data is stored in an SQLite database.

Why is my first request slow?

The first request when you type something in the search box or click on the RANDOM button might be slow. Please refer to this link Free Web Services
